HTML

History

HTML is the foundation of the webpage. It tells us what content is text video and image. It started from HTML1.0 to HTML5, it got advanced every year. It was in 1991.It was designed by Tim Berners-Lee

HTML Tags

They tell the browser what each part of the page is and how it should be structured

  1. Comments

    2. HTML comments are not displayed in the browser, but they can help document your HTML source code.

  2. Forms

    3. An HTML form is used to collect user input. The user input is most often sent to a server for processing.

  3. Canvas

    4. The HTML element is used to draw graphics on a web page.

  4. Iframes

  5. HTML iframe is used to display a web page within a web page.

CSS(Cascading Style Sheet)

CSS is the language we use to style an HTML document. CSS describes how HTML elements should be displayed

CSS properties and techniques

  1. Flexbox

    2. Aligns and distributes items evenly for responsive design

  2. Color and background-color

    3. Defines text and background colors for a clean theme

  3. Hover

    4. Adds intercativity by changing an element's style when the mouse hovers over it (button color change)

  4. Font-family

  5. Control typography and improves readability
